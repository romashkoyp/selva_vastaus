About this website

"Selvä vastaus", which means "Clear answer", is a service for individuals interested in studying the Finnish language. Here, you can discover Finnish words that are often difficult to find elsewhere.

I created it for my own purposes because I am simultaneously studying Finnish and IT. This website is my final project for the "CS50's Introduction to Computer Science" free course at Harvard University.

When I study Finnish grammar, I need to know word endings every time. Understanding how Finnish words can flex in different cases is crucial. For instance, take a look at the table below.

I am open to your suggestions and recommendations to improve the website. You can contact me by email at romashkoyp(a)gmail.com

Source of information

To create a database and verify Finnish words and their types, I used several sources:

— Wikisanakirja - to find all nouns, verbs, adjectives, monosyllabic, and two-syllable words

— kieli.net - to find translations and check specific cases of words

— bab.la - to find translations

— The Language Bank of Finland - to find frequently written words and all Finnish words in nominative form

— my own algorithms for word validation: old and new i-words, short and long words, hauska- and koira-type words

Technical information

This website showcases a web application built with Python, Flask, Jinja, and SQLite for the backend, and HTML, CSS, JavaScript, and Bootstrap for front-end rendering. Basic functions include:

— input validation on both the front-end and server

— retention of user input in the input field during the user's session

— construction of SQL queries by the user based on selected criteria and execution using parameters to fetch word data

— display of search results using HTML templates

— independent handling of GET and POST requests by the website

— the website automatic adaptation to ensure compatibility and optimal layout on both desktop and mobile devices
